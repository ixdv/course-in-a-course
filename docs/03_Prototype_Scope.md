# Prototype Scope for RoomLight

## 1. Prototype Description
This prototype is a simple Python program running in a terminal. Its built for hotel managers to show how they can control many lights from one place. The program shows that an admin can set up lighting for the whole hotel sitting at a desk instead of walking to every room. It shows the sync everywhere idea works fast and effortlessly.

## 2. Requirements the prototype will prove
The prototype will prove that these requirements from my list can be built:

* **REQ-001 & REQ-002 (Syncing):** Prototype will show how an admin creates a light setting and sends it to many virtual rooms at the same time
* **REQ-003 (Status List):** It prints a list of rooms to show their light status so the admin can check the whole hotel at once
* **REQ-007 (Local Memory):** The code will show that each virtual room remembers its own setting, so the lights stay on even if the main program is turned off
