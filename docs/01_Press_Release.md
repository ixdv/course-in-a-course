# RoomLight - The Smart Light Controller: Easy light management dashboard for hotels

Team Insomnia announced today a new professional lighting system — RoomLight — a dashboard that allows hotel managers and customers to adjust the lights with the touch of a finger.

Everyone knows the struggle: "Why are there so many light switches and why are the lights so bright?" For customers this can be frustrating and for the staff managing hotel lights can be time consuming on a room-by-room basis. Roomlight fixes the inconsistency. The system allows you to set a lighting profile on an admin dashboard and sync it globally to all the hotel rooms. Guests will also be able to adjust their own rooms lightning during the visit to their liking.

"We wanted to create a tool that removes all the manual labor of constantly adjusting smart lights", said the Lead Developer at Team Insomnia. "By focusing on central management by admins and individual room adjustments for guests we can keep the lighting standard for each visitor while allowing them freedom to adjust to their liking."

RoomLight is easy to setup with your existing network. Just install the controller on a server or computer thats always on, log into the web dashboard, and start adding lights. For customers you can create rooms and set up tablets that have access to controlling that specific rooms lights.

"Adjusting smart lighting based on seasons used to take days of walking through every room. With RoomLight, I set up lighting in the whole hotel from my desk in five minutes", said a Manager after their first week of use.

FAQ
- How many rooms can it sync? From 1 to 500+ rooms simultaneously from the web dashboard
- Can guests still change their own rooms lights? Yes! The syncing profile will only set the default lighting but guests can adjust their own room to their liking.
- Is it hard to use? No, the web gui is easy to navigate with simple and clear buttons for setting the lights.
- What if the hotel network goes down? Each rooms lighting is stored locally so lights will keep the latest setting even without a connection.
